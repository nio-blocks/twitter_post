TwitterFavorite
===============
Favorite tweets on Twitter.

Properties
----------
- **creds**: Twitter api app credentials.
- **id**: Identifier for tweet to favorite.

Inputs
------
- **default**: Any list of signals.

Outputs
-------
None

Commands
--------
None
